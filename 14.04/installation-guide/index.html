Installing Ubuntu 14.04

This site is where you can find the official documentation developed and maintained by the Ubuntu Documentation Project.

This page contains documentation for Ubuntu 14.04 LTS, the latest LTS, released in April 2014.

Here you can find detailed information for a variety of methods for installing Ubuntu. More advanced installation options are covered. For a standard Desktop CD installation, which is the recommended method of installing Ubuntu, the Graphical Install page on the Ubuntu community documentation site might be the preferred reference. Also, there is the more general Ubuntu community installation site.

To view the detailed installation guide, choose the architecture of your computer and your desired delivery format from the list below.

Maintenance of this installation guide has fallen behind. Some information can be misleading and/or obsolete.
