3.3. Point release and HWE kernel

The LTS release’s kernel is refreshed with a new, optional kernel at some point releases. This kernel is referred to as the Hardware Enablement (HWE) kernel, and provides support for new hardware that has been introduced after the LTS’s GA. New hardware components are supported by HWE kernels and on an installed system with proper networking you can install it by an apt command which you can find it at LTSEnablementStack on Ubuntu wiki.

The LTS point releases also provide an updated installation kernel for scenarios where kernel support for key components, such as NIC or RAID cards, is required during the installation process.
